Medium: Fizz Buzz

Write a function that, when given a whole number, checks it's remainder when divded by 3 or 5 and outputs a relevant string

  • Function should recieve an integer
  • If integer is divisible by 3, return "Fizz" string
  • If integer is divisible by 5, return "Buzz" string
  • If integer is divisible by both, return "FizzBuzz" string
  • Return output to "m-out" p tag
